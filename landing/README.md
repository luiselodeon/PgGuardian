# PgGuardian — Landing Page

Landing page de PgGuardian, un servicio de auditoría automatizada para PostgreSQL.

## Stack

- React 18
- Vite
- Tailwind CSS

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
